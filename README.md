# AI Klinik - Sprint 1

Bu proje, klinik vaka simülasyon platformunun temel altyapısını ve kullanıcı arayüzünü içerir. Flask tabanlıdır.

## Kurulum

1. Sanal ortam oluşturun ve aktif edin:
   ```
   python -m venv venv
   venv\Scripts\activate
   ```
2. Gerekli paketleri yükleyin:
   ```
   pip install -r requirements.txt
   ```
3. Uygulamayı başlatın:
   ```
   python run.py
   ```

## Özellikler (Sprint 1)
- Kullanıcı girişi ve oturum açma
- Ana sayfada vaka listesi
- Vaka detay sayfası
- Temel backend (Flask + SQLite)
- Temiz ve anlaşılır arayüz (Bootstrap)
- Veritabanı şeması ve veri saklama
